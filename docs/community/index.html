Community

Mailing Lists

End-Users should subscribe the the Libk-users mailing list.

Developers interested in extending and enhancing Libki should join the Libki-developers mailing list.

Documentation

For client installation, please visit the client wiki.

For server installation, check out the server README and the server wiki on GitHub.

Issues and bugs

If you have issues, please file them on the GitHub Libki client bug tracker or server bug tracker.

Development

The canonical source for Libki source code is currently GitHub. Please make all pull requests via GitHub.
